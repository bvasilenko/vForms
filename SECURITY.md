# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x | Yes |

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Report via email: security@booga.dev

Include:
- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Potential impact

Response within 72 hours. Patch releases follow responsible disclosure — public disclosure coordinated after a fix is available.
